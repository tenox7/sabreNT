Sabre Fighter Plane Simulator
0.2.4.aws patch for Win NT/95/98.

Thanks for trying out Sabre Fighter Plane
Simulator. This is an patch to the 0.2.3*
Windows distribution. Please note that this
version requires DirectX 5.2 and up.

For this patch to work, you must have installed 
the full version of Sabre, available at 
http://sabre.cobite.com. In addition, you
must have DirectX 5.2+ installed.


WHATS NEW

Sound, finally! Explosions, engine sounds, gun fire and
even wingmen chatter. Drawback is you now need to have
DirectX 5.2 or better installed to run Sabre.

Joystick support, including throttle wheel and
rudder pedals if you have 'em. Menu selection
"Controls/Devices ..." brings up dialog so you 
can select joystick, mouse & keyboard combos.

New views, including a virtual cockpit and
padlock.

Some key commands have been changed, check the
help topic "Keys Reference" for updated listing.

Enjoy!

Dan Hammer
