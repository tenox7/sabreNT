Sabre Fighter Plane Simulator
0.2.4.b.ws patch for Win NT/95/98/2000.
November 20, 1999

Thanks for trying out Sabre Fighter Plane
Simulator. This is an patch to the 0.2.4.a.ws
Windows distribution, introducing some minor
changes.

For this patch to work, you must have installed 
the full version of Sabre 0.2.4.a.ws. Please also 
note this program requires DirectX 6.0+ for sound
support. This installation *does not* install 
DirectX for you. Contact Microsoft for latest 
DirectX drivers. 



WHATS NEW

Joystick hat control fixed
Flight model refinement by Pierre de Mouveaux
2 Wingmen accompany you on "fly" mission
Wingmen start out in proper formation

Enjoy!

Dan Hammer
11/21/99
